# 🧊 ClassiCube AIO (All-in-One)

[![License](https://img.shields.io/badge/License-BSD%203--Clause-blue.svg)](https://opensource.org/licenses/BSD-3-Clause)
[![Size](https://img.shields.io/badge/Size-Ultralight-brightgreen.svg)]()
[![Platform](https://img.shields.io/badge/Platform-Web%20Browser-orange.svg)]()

**Minecraft Classic, but make it fit in your pocket.** This repository contains a highly optimized, modified, and completely self-contained version of **ClassiCube**. I’ve stripped away the bulk and bundled everything into a **single `.html` file**. No installers, no folders, no "where did that DLL go?"—just pure, blocky nostalgia that runs on a toaster.

---

## ✨ Key Features

* **🚀 Zero Installation:** Open the `.html` file in any modern browser and start building.
* **📦 All-in-One:** Assets, scripts, and logic are bundled into one lightweight file.
* **⚡ High Performance:** Modified to be exceptionally "snappy," even on lower-end hardware.
* **🌐 Browser-Native:** Optimized for WebGL and modern JS engines.
* **💾 Offline Ready:** Save the file to your desktop and play it without an internet connection.

## 🔧 Technical Modifications

This version isn't just a copy-paste. To achieve the "All-in-One" magic, the following was done:
* **Asset Inlining:** Base64 encoding for critical textures to reduce HTTP requests.
* **Script Minification:** Reduced the footprint of the engine while maintaining compatibility.
* **UI Tweaks:** Cleaned up the interface for a better browser-focused experience.
